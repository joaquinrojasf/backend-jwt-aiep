# Proyecto Backend - Autenticación con JWT

## Descripción
Este proyecto implementa un servidor backend con Node.js y Express.js para autenticar usuarios mediante JWT, almacenar el token en cookies y proteger rutas privadas con middleware.

## Tecnologías
- Node.js
- Express.js
- JSON Web Token
- Cookie-parser
- GitHub
- Postman

## Funcionalidades
- Login con usuarios ficticios
- Generación de token JWT
- Almacenamiento del token en cookie
- Ruta protegida con middleware
- Logout
